# /update-progress - セッション終了儀式

## やること

1. 今セッションで行った作業を振り返る
2. @progress.md を更新:
   - Current objective
   - Recent commands executed
   - Known issues / blockers
   - Notes

## 注意

- 「次やること」は progress.md に書かない（TASKS.md に統一）
- TASKS.md の Done 移動は人間が確認してから

## 出力フォーマット

```
## Session End Summary

- **Completed**: 完了したこと
- **In Progress**: 途中のこと
- **Blockers**: 発見した問題

progress.md を更新しました。
```
